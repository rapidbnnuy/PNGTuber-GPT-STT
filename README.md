# PNGTuber-GPT-STT

WebGPU-accelerated Speech-to-Text for PNGTuber-GPT.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [Git](https://git-scm.com/)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rapidbnnuy/PNGTuber-GPT-STT.git
    cd PNGTuber-GPT-STT
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running Locally

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

### Windows VM Notes

If you are running this on a Windows VM:
- Ensure you have a browser that supports WebGPU installed (e.g., Chrome, Edge).
- You may need to enable hardware acceleration in your VM settings if WebGPU performance is poor.
